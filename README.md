# Synthetic Identity Fraud Detection Using Graph Analytics and Behavioral Risk Scoring

## Overview
This repository contains a reusable AI-driven synthetic identity fraud detection framework using graph analytics, behavioral risk scoring, entity resolution, machine learning, and real-time fraud decisioning.

The framework is designed for banks, fintech platforms, digital lenders, payment providers, credit bureaus, and enterprise fraud intelligence teams.

## Business Problem
Synthetic identity fraud occurs when fraudsters create fabricated identities using a mix of legitimate and false information, such as valid Social Security Numbers combined with fake names, addresses, phones, employment details, devices, or emails.

Traditional rule-based systems often fail because synthetic identities can appear legitimate during onboarding and early account activity.

## Solution Objectives
- Detect hidden relationships across identities, devices, accounts, addresses, and transactions.
- Identify synthetic fraud rings using graph analytics.
- Score identity risk using behavioral and transactional indicators.
- Detect suspicious account clusters and shared attributes.
- Support real-time fraud decisioning.
- Provide explainable reason codes for investigators.

## Architecture Layers
1. Data ingestion
2. Identity resolution
3. Entity graph construction
4. Behavioral feature engineering
5. Graph analytics
6. Machine learning risk scoring
7. Real-time decision engine
8. Fraud analyst workflow
9. Explainability and governance
10. Monitoring and drift detection
